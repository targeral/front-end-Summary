# Summary
## 有意义的标记
* h1, h2等;
* ul, ol和dl;
* strong和em;
* blockquote和cite;
* abbr, acronym和code
* fieldset,length和label

## 为元素命名(Id和类名)
**根据“它们是什么”来为元素命名**

## 使用类还是Id
**类应该应用于概念上相似的元素，这些元素可以出现在同一个页面上的多个位置，而ID应该应用于不同的唯一的元素。**

*不要滥用类，而要学会“组件化”它们*

## div和span
**不要滥用，也不要畏惧**
